## Overview

The Achievements & Scoring System rewards players for:
- **Exploration**: Finding secrets, evidence, hidden rooms
- **Mastery**: Clearing hazards, completing QTEs, saving NPCs
- **Survival**: Escaping levels, managing fear/health
- **Collection**: Gathering all evidence from a disaster
- **Narrative**: Making specific story choices

### Design Philosophy

**Achievements should:**
- Reward skill and exploration (not just completion)
- Encourage multiple playthroughs (varied approaches)
- Respect player time (not grindy)
- Tell stories (meaningful names and descriptions)
- Balance common vs. rare (some easy, some challenging)

**Achievements should NOT:**
- Be impossible (unfair requirements)
- Contradict each other (mutually exclusive)
- Require random chance (luck-based)
- Spoil narrative (hidden achievements for twists)

### Achievement Rarity

| Rarity | % Players | Examples |
|--------|-----------|----------|
| **Common** | 60-90% | Complete Level 1, Unlock first room |
| **Uncommon** | 30-60% | Save one NPC, Clear 3 hazards |
| **Rare** | 10-30% | Collect all evidence, Save all NPCs |
| **Epic** | 3-10% | Complete game without dying, Perfect QTE streak |
| **Legendary** | <3% | Speedrun, No fear gained, Secret ending |

## Achievement Anatomy

### File Location
`data/player_achievements.json`

### Minimal Required Schema
```json
{
  "achievement_id": {
    "name": "Achievement Name",
    "description": "What the player accomplished",
    "unlock_conditions": {
      "type": "value"
    }
  }
}
```

### Complete Schema
```json
{
  "achievement_id": {
    "name": "Display Name",
    "description": "Detailed description",
    "hidden": false,
    "hidden_description": "??? - Complete a secret objective",
    "category": "exploration",
    "rarity": "uncommon",
    "icon": "icon_filename",
    "score_reward": 50,
    "unlock_conditions": {
      "hazards_cleared": 3,
      "npcs_saved": 1,
      "items_collected": ["item_key"],
      "evidence_collected": {
        "source": "disaster_name",
        "count": 3
      },
      "rooms_visited": 10,
      "level_completed": 1,
      "qte_success_streak": 3,
      "no_deaths": true,
      "fear_below": 30,
      "health_above": 70,
      "turns_under": 50,
      "special_action": "secret_ending"
    },
    "unlock_message": "Achievement Unlocked: {name}!",
    "prerequisites": ["other_achievement_id"],
    "unlocked": false,
    "unlock_timestamp": null,
    "progress": {
      "current": 0,
      "target": 3,
      "trackable": true
    }
  }
}
```

### Field Descriptions

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | ✅ Yes | Display name (e.g., "The Visionary") |
| `description` | string | ✅ Yes | What player accomplished |
| `hidden` | boolean | ❌ No | Whether achievement is secret (default: false) |
| `hidden_description` | string | ❌ No | Placeholder text before unlock |
| `category` | string | ❌ No | exploration, mastery, survival, collection, narrative |
| `rarity` | string | ❌ No | common, uncommon, rare, epic, legendary |
| `icon` | string | ❌ No | Icon filename (future feature) |
| `score_reward` | int | ❌ No | Points awarded (default: 50) |
| `unlock_conditions` | object | ✅ Yes | Conditions to unlock (see Unlock Conditions) |
| `unlock_message` | string | ❌ No | Custom message on unlock |
| `prerequisites` | array | ❌ No | Achievement IDs required first |
| `unlocked` | boolean | ❌ No | Runtime state: has player unlocked? |
| `unlock_timestamp` | string | ❌ No | Runtime: when unlocked |
| `progress` | object | ❌ No | Trackable progress (current/target) |

## Achievement Categories

### Exploration

**Focus:** Discovering secrets, visiting all areas, finding hidden content.

**Examples:**
- "Cartographer": Visit all rooms in Level 1
- "Secrets of the Hospital": Find all hidden items
- "Off the Beaten Path": Discover a secret room

**Unlock Conditions:**
- `rooms_visited: X`
- `secret_rooms_found: X`
- `items_collected: [...]`

---

### Mastery

**Focus:** Skill-based challenges, hazard clearing, QTE performance.

**Examples:**
- "Defusal Expert": Clear 5 hazards without failing
- "Perfect Timing": Complete 3 QTEs in a row without errors
- "Calm Under Pressure": Clear MRI hazard on first try

**Unlock Conditions:**
- `hazards_cleared: X`
- `qte_success_streak: X`
- `qte_perfect_count: X`

---

### Survival

**Focus:** Staying alive, managing resources, minimizing damage.

**Examples:**
- "Survivor": Complete Level 1 without dying
- "Fearless": Finish with fear below 30%
- "Hardy": Complete level with health above 70%

**Unlock Conditions:**
- `no_deaths: true`
- `fear_below: X`
- `health_above: X`

---

### Collection

**Focus:** Gathering evidence, finding all items of a type.

**Examples:**
- "Train Case Closed": Collect all train derailment evidence
- "Evidence Collector": Gather 10 pieces of evidence
- "Key Master": Collect all keys in Level 1

**Unlock Conditions:**
- `evidence_collected: {source: "...", count: X}`
- `items_collected: [...]`
- `item_type_count: {type: "evidence", count: X}`

---

### Narrative

**Focus:** Story choices, NPC interactions, special endings.

**Examples:**
- "The Visionary": Obtain Corbin's Journal
- "Hero of the Hospital": Save all NPCs in Level 1
- "Design Breaker": Unlock the secret ending

**Unlock Conditions:**
- `npcs_saved: X`
- `special_action: "..."`
- `items_collected: ["quest_item"]`

## Unlock Conditions

Achievements unlock when all specified conditions are met.

### Single Condition Types

#### Hazards Cleared
```json
{
  "unlock_conditions": {
    "hazards_cleared": 3
  }
}
```
**Behavior:** Unlock when player successfully clears 3 hazards (terminal success state).

---

#### NPCs Saved
```json
{
  "unlock_conditions": {
    "npcs_saved": 1
  }
}
```
**Behavior:** Unlock when 1 NPC reaches "saved" fate.

---

#### Items Collected
```json
{
  "unlock_conditions": {
    "items_collected": ["corbins_journal", "patient_chart"]
  }
}
```
**Behavior:** Unlock when player has collected ALL listed items.

---

#### Evidence Collection (Specific Source)
```json
{
  "unlock_conditions": {
    "evidence_collected": {
      "source": "train_derailment",
      "count": 3
    }
  }
}
```
**Behavior:** Unlock when player has collected 3 evidence items from "train_derailment" disaster.

---

#### Evidence Collection (Any Source)
```json
{
  "unlock_conditions": {
    "evidence_count": 10
  }
}
```
**Behavior:** Unlock when player has collected 10 evidence items total (any source).

---

#### Rooms Visited
```json
{
  "unlock_conditions": {
    "rooms_visited": 12
  }
}
```
**Behavior:** Unlock when player has visited 12 unique rooms.

---

#### Level Completed
```json
{
  "unlock_conditions": {
    "level_completed": 1
  }
}
```
**Behavior:** Unlock when player completes Level 1 (reaches level exit).

---

#### QTE Success Streak
```json
{
  "unlock_conditions": {
    "qte_success_streak": 3
  }
}
```
**Behavior:** Unlock when player completes 3 QTEs in a row without failure.

---

#### QTE Perfect Count
```json
{
  "unlock_conditions": {
    "qte_perfect_count": 5
  }
}
```
**Behavior:** Unlock when player completes 5 QTEs with zero errors (perfect performance).

---

#### No Deaths
```json
{
  "unlock_conditions": {
    "no_deaths": true
  }
}
```
**Behavior:** Unlock when player completes level without dying (no Game Over screens).

---

#### Fear Threshold
```json
{
  "unlock_conditions": {
    "fear_below": 30
  }
}
```
**Behavior:** Unlock when player completes level with fear ≤ 30%.

---

#### Health Threshold
```json
{
  "unlock_conditions": {
    "health_above": 70
  }
}
```
**Behavior:** Unlock when player completes level with health ≥ 70%.

---

#### Turn/Time Limit
```json
{
  "unlock_conditions": {
    "turns_under": 50
  }
}
```
**Behavior:** Unlock when player completes level in under 50 turns (speedrun).

---

#### Special Action
```json
{
  "unlock_conditions": {
    "special_action": "secret_ending_unlocked"
  }
}
```
**Behavior:** Unlock when specific narrative trigger fires (custom event).

### Multiple Conditions (AND Logic)

All conditions must be met.

```json
{
  "hero_of_the_hospital": {
    "name": "Hero of the Hospital",
    "description": "Save all NPCs and clear all hazards in Level 1 without dying.",
    "unlock_conditions": {
      "npcs_saved": 3,
      "hazards_cleared": 5,
      "no_deaths": true,
      "level_completed": 1
    }
  }
}
```

**Behavior:**
- Player must save 3 NPCs
- AND clear 5 hazards
- AND not die
- AND complete level

---

### Prerequisites

Achievement requires another achievement first.

```json
{
  "master_evidence_collector": {
    "name": "Master Evidence Collector",
    "description": "Collect all evidence from all disasters.",
    "prerequisites": [
      "train_case_closed",
      "plane_case_closed",
      "hotel_case_closed"
    ],
    "unlock_conditions": {
      "evidence_count": 15
    }
  }
}
```

**Behavior:**
- Player must first unlock "train_case_closed", "plane_case_closed", "hotel_case_closed"
- THEN collect 15 total evidence items
- Only then will "master_evidence_collector" unlock

## Scoring System

### Score Sources

| Source | Points | Frequency |
|--------|--------|-----------|
| **Hazard Cleared** | 50-150 | Per hazard |
| **QTE Success** | 10-30 | Per QTE |
| **NPC Saved** | 100 | Per NPC fate: saved |
| **NPC Died** | -25 to -75 | Per NPC fate: died/abandoned |
| **Evidence Collected** | 25-50 | Per evidence item |
| **Achievement Unlocked** | 50-200 | Per achievement |
| **Level Completed** | 200-500 | Per level |
| **Item Found** | 5-20 | Per item taken |
| **Death Penalty** | -100 | Per death |

---

### Score Calculation

**Base Formula:**
```python
total_score = (
    hazards_cleared_score +
    qte_success_score +
    npc_fate_score +
    evidence_score +
    achievement_score +
    level_completion_score +
    item_score -
    death_penalty
)
```

### Common Issues

#### **Achievement Not Unlocking**
**Symptoms:** Player meets conditions but achievement doesn't unlock  
**Causes:**
- Condition syntax error
- Game state doesn't match expected values
- Prerequisites not met
- Achievement already unlocked (duplicate prevention)

**Debug:**
```python
# Check unlock conditions:
achievement["unlock_conditions"]  # Verify syntax and values

# Check game state:
player.npcs_saved_count  # Should match unlock_conditions["npcs_saved"]
player.hazards_cleared_count  # Should match unlock_conditions["hazards_cleared"]

# Check prerequisites:
achievement["prerequisites"]  # All must be unlocked first

# Check unlock status:
achievement["unlocked"]  # Should be False before unlock
```

---

#### **Score Not Adding Correctly**
**Symptoms:** Score doesn't increase or decreases unexpectedly  
**Causes:**
- Negative score sources (death penalty, NPC fates)
- Score already awarded (duplicate)
- Score calculation bug

**Debug:**
```python
# Check score sources:
# Log each score addition/subtraction
print(f"Score: +{amount} for {reason} (total: {total_score})")

# Check NPC fates:
npc["current_fate"]  # "died_hazard" = -50, "saved" = +100

# Check death count:
player.death_count  # Each death = -100
```

---

#### **Progress Not Tracking**
**Symptoms:** Achievement progress bar doesn't update  
**Causes:**
- `trackable: false` or missing
- Progress not incremented in code
- UI not refreshing

**Debug:**
```python
# Check trackable flag:
achievement["progress"]["trackable"]  # Should be True

# Check progress values:
achievement["progress"]["current"]  # Should increment
achievement["progress"]["target"]  # Should match unlock_conditions

# Manual update test:
achievement["progress"]["current"] += 1
print(f"Progress: {current} / {target}")
```

---

#### **Hidden Achievement Revealed**
**Symptoms:** Hidden achievement shows name/description before unlock  
**Causes:**
- `hidden: false` or missing
- UI not checking hidden flag
- Achievement unlocked (should now be revealed)

**Debug:**
```python
# Check hidden flag:
achievement["hidden"]  # Should be True

# Check unlock status:
achievement["unlocked"]  # If True, should show full details

# Check UI logic:
if achievement["hidden"] and not achievement["unlocked"]:
    display(achievement["hidden_description"])
else:
    display(achievement["description"])
```

- [ ] Achievement ID is unique

### Example 2: Complex Achievement (Mastery)

**Perfect Timing**

**Achievement Definition:**
```json
{
  "perfect_timing": {
    "name": "Perfect Timing",
    "description": "Complete 3 QTEs in a row without any mistakes.",
    "category": "mastery",
    "rarity": "rare",
    "score_reward": 150,
    "unlock_conditions": {
      "qte_success_streak": 3
    },
    "progress": {
      "current": 0,
      "target": 3,
      "trackable": true
    },
    "unlocked": false
  }
```

**Unlock Logic:**
```python
# When QTE succeeds:
if qte_errors == 0:
    player.qte_success_streak += 1
    achievements["perfect_timing"]["progress"]["current"] = player.qte_success_streak
    
    if player.qte_success_streak >= 3:
        unlock_achievement("perfect_timing")
else:
    player.qte_success_streak = 0
    achievements["perfect_timing"]["progress"]["current"] = 0

# When QTE fails:
player.qte_success_streak = 0
achievements["perfect_timing"]["progress"]["current"] = 0
```

**Unlock Message:**
```
🏆 Achievement Unlocked: Perfect Timing!

Complete 3 QTEs in a row without any mistakes.

Your reflexes are impeccable. Death's traps
can't catch you off-guard.

+150 points
```
