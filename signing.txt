## **4. Sign the AAB**

### **Option A: Interactive (Prompts for Password)**

```bash
jarsigner -verbose \
  -sigalg SHA256withRSA \
  -digestalg SHA-256 \
  -keystore dep-fdt-release.keystore \
  bin/de_fdt-x.x.x-arm64-v8a_armeabi-v7a-release.aab \
  de_fdt
```

jarsigner -verbose   -sigalg SHA256withRSA   -digestalg SHA-256   -keystore dep-fdt-release.keystore   bin/dep_fdt-0.2.12-arm64-v8a_armeabi-v7a-release.aab   dep_fdt

You'll be prompted:
```
Enter Passphrase for keystore: [TYPE PASSWORD]
Enter key password for dep_fdt: [TYPE PASSWORD or ENTER if same]
```

### **Option B: Non-Interactive (Password in Command)**

```bash
jarsigner -verbose \
  -sigalg SHA256withRSA \
  -digestalg SHA-256 \
  -keystore dep-fdt-release.keystore \
  -storepass YOUR_KEYSTORE_PASSWORD \
  -keypass YOUR_KEY_PASSWORD \
  bin/dep_fdt-0.2.2-arm64-v8a_armeabi-v7a-release.aab \
  de_fdt
```

### **Option C: Secure Script (Recommended)**

Create a script:
```bash
nano sign-aab.sh
```

Add this content:
```bash
#!/bin/bash
# sign-aab.sh - Signs AAB with keystore

AAB_FILE="bin/dep_fdt-0.2.2-arm64-v8a_armeabi-v7a-release.aab"
KEYSTORE="dep-fdt-release.keystore"
ALIAS="dep_fdt"

# Check AAB exists
if [ ! -f "$AAB_FILE" ]; then
    echo "Error: AAB not found at $AAB_FILE"
    exit 1
fi

# Check keystore exists
if [ ! -f "$KEYSTORE" ]; then
    echo "Error: Keystore not found at $KEYSTORE"
    exit 1
fi

# Prompt for passwords securely
read -sp "Enter keystore password: " KEYSTORE_PASS
echo
read -sp "Enter key password (or press ENTER if same as keystore): " KEY_PASS
echo

# Use keystore password if key password is empty
if [ -z "$KEY_PASS" ]; then
    KEY_PASS="$KEYSTORE_PASS"
fi

# Sign the AAB
echo "Signing $AAB_FILE..."
jarsigner -verbose \
  -sigalg SHA256withRSA \
  -digestalg SHA-256 \
  -keystore "$KEYSTORE" \
  -storepass "$KEYSTORE_PASS" \
  -keypass "$KEY_PASS" \
  "$AAB_FILE" \
  "$ALIAS"

if [ $? -eq 0 ]; then
    echo "✓ AAB signed successfully!"
    
    # Verify signature
    echo "Verifying signature..."
    jarsigner -verify -verbose -certs "$AAB_FILE"
    
    if [ $? -eq 0 ]; then
        echo "✓ Signature verified!"
    else
        echo "✗ Signature verification failed!"
        exit 1
    fi
else
    echo "✗ Signing failed!"
    exit 1
fi
```

Make it executable and run:
```bash
chmod +x sign-aab.sh
./sign-aab.sh
```

## **8. Update [`buildozer.spec`](buildozer.spec ) to Auto-Sign**

Your buildozer.spec already has the keystore configured! Just provide passwords via environment variables:

```bash
# Create a build script
nano build-release.sh
```

Add:
```bash
#!/bin/bash
# build-release.sh - Build and sign AAB automatically

read -sp "Enter keystore password: " KEYSTORE_PASS
echo
export KEYSTORE_PASS
export KEY_PASS="$KEYSTORE_PASS"  # or prompt separately if different

# Build release AAB (buildozer will sign it automatically)
buildozer android release

echo "✓ Build complete! AAB signed with keystore."
```

Make executable:
```bash
chmod +x build-release.sh
```

Future builds:
```bash
./build-release.sh
```
